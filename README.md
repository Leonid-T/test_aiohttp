# test_aiohttp_junior

CRUD api на таблицу пользователей с правами доступа и авторизацией.

Использованные технологии:
- Aiohttp
- SqlAlchemy core
- Postgresql
- Docker
- Pytest

## Установка и запуск

- запуск: make up
- документация: http://localhost:8080/backend

## Запуск тестов

- создать тестовую базу
- изменить test_db_url в файле tests/fixtures.py
